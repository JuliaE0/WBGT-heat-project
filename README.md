# WBGT-heat-project

Data used:
- ERA5-Land hourly data from 1950 to present (https://cds.climate.copernicus.eu/datasets/reanalysis-era5-land?tab=overview)
- ERA5 hourly data on single levels from 1940 to present (https://cds.climate.copernicus.eu/datasets/reanalysis-era5-single-levels?tab=overview)

heatmetrics package from https://figshare.com/articles/software/heatmetrics_R_Package/19739965?file=36838296
- This package works on high-resolution gridded reanalysis data.

Spangler, Keith (2022). heatmetrics R Package. figshare. Software. https://doi.org/10.6084/m9.figshare.19739965.v2
